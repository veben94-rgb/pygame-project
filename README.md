# PyGame Практическая работа

## Описание
Игра создана с использованием PyGame и SQLite3.

Игрок должен собирать монеты и избегать врагов.

## Возможности
- Управление стрелками
- Подсчёт очков
- Таблица рекордов
- SQLite база данных
- Победа и поражение
- ООП

## Запуск:

1. pip install pygame

2. python main.py
